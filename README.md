# 🚀 AI Startup Command Center

An end-to-end **Business Intelligence and Machine Learning** project that transforms startup data into actionable executive insights using **Power BI, Python, and Machine Learning**.

---

## 📌 Project Overview

The AI Startup Command Center is a multi-page analytics platform designed to help startup leadership monitor:

- 📈 Revenue Performance
- 👥 Customer Intelligence
- 📣 Marketing Performance
- 🤖 AI-Powered Churn Prediction

The project combines business intelligence techniques with machine learning to provide data-driven decision-making capabilities.

---

## 🛠️ Tech Stack

- Power BI
- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning (Random Forest)
- Git & GitHub
- CSV Datasets
